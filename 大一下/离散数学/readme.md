老师是lym，人很好，讲课也还可以，问问题会耐心解答。这门课是双语教学，教材、ppt均为英文，期中期末考试试卷也是英文。对于答题老师说尽量写英文，但是不会因为单词语法错误强行扣分。

分数构成是平时30期中10期末60。期中考试10道大题，随堂考90分钟。期末10道选择，10道填空，6道大题.

平时作业是交纸质版，每一章交一次作业，下一次课会批改完后发回来。**建议交上去之前和发下来之后第一时间拍照存档**。lym 老师平时作业给分比较松，认真完成都有95+，请一定一定认真完成课后作业，建议做完之后对一下答案，不会的抄答案也交上去。

期末不是特别难，把ppt弄懂即可。最后一张语言与自动机有点抽象，并且大概率会有一道大题，建议多花时间弄清楚，这部分与数电有相似的地方。ppt上的每道例题都要会写（包括一些定理的证明，例如证明德摩根律、证明素数的无限性等）。建议期末前花点时间看看术语的中英文，避免看不懂题（如果是lym 老师班，遇到看不懂的单词可以举手问一下）。

这门课可以考前速成，花两三天过一下ppt问题不大。如果想考高分，建议平时每一章学完后复习一下ppt，并且多做一些课后练习（英文书课后练习很多，量大管饱）。
