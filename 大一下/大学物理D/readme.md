这门课难度比较大，4学分占比也不小。课程内容很多，一个学期要学完大物上下两本书，知识量大。不过大部分是高中物理的框架，加上微积分。注意上课认真听，ppt上的题目全部弄懂。占比是平时分20，期中20，期末60。24级期中是25道选择题，期末是20选择题+4大题。

学习建议：

1. 认真完成课后作业和ppt例题，期中大概率会出现原题。有时间最好做一下课本配套的题目。课本配套有一本习题解析。
2. 一定学好积分、二重积分，基本计算题都要用到。
3. 可以搭配网课学习，推荐网课：[大学物理 1-1-质点运动的描述1-参考系-质点-位失-运动方程-位移_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV17i4y1D7HK/?spm_id_from=333.1387.homepage.video_card.click&vd_source=71f03209a5b313c45551673bc0ba3c0f)
4. **建议期末前两天，要刷一遍课本和习题的选择题，大概率有部分原题**
5. 大物不建议速通，建议提前三周开始复习。
